# kubetools

Simple nodejs scripts to replace some of the functionality of kubectl.

apply.js ->  apply yaml files to a deployment
describe.js -> describe a pod
editdeployment.js -> patch a deployment
exec.js ->  shell to a pod
getevents.js ->  get events 
getnamespaces.js ->  get the list of namespace in current context
getpods.js ->  get the pods in a namespace
scaledeployment.js ->  set replica count for a deployment
